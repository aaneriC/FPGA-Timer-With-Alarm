# FPGA-Timer-With-Alarm

This repository contains my project that utilizes Verilog code with the NEXYS A7-100t FPGA and Vivado.
I incorporated the on-board 7-Segment displays, audio, switches, and LED in order to create a fully functional system that allows a user to set the time and from there count upwards. 
Once the timer reaches the user-set time limit, the LEDs will flash and an audio will play.
